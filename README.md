# Reproductor de Música Estética 🎧

Un reproductor de música web con diseño moderno y minimalista, optimizado para una experiencia visual "Aesthetic". Permite sintonizar una radio de trading en vivo o cargar tus propios archivos MP3 locales.

## ✨ Características
- **Modo Radio Live**: Sintonización directa de estaciones de trading y Lo-Fi.
- **Soporte MP3 Local**: Carga y reproduce tus propios archivos de música desde tu dispositivo.
- **Interfaz Estética**: Animaciones suaves, controles de volumen personalizados y diseño adaptable.
- **Iconografía**: Uso de FontAwesome para una navegación intuitiva.

## 🛠️ Tecnologías
- HTML5 (Estructura Semántica)
- CSS3 (Variables, Flexbox y Animaciones)
- JavaScript Vanilla (Lógica de Audio y Manejo de Archivos)
- [FontAwesome](https://fontawesome.com/) (Iconos)

## 🚀 Cómo usar
1. Abre el archivo `index.html` en tu navegador.
2. Selecciona **Radio Live** para escuchar el flujo de streaming configurado.
3. Cambia a **MP3 Local** si deseas subir una canción de tu carpeta personal.
